# 🍱 Restaurant Order System

Sistema completo de pedidos online para restaurantes com painel administrativo.

Este sistema permite que clientes façam pedidos diretamente pelo celular enquanto o restaurante acompanha os pedidos em tempo real.

---

# 🚀 Funcionalidades

## Cliente

- Visualização do cardápio
- Adicionar itens ao carrinho
- Cálculo automático do total
- Informações de entrega
- Envio do pedido

## Painel administrativo

- Visualizar pedidos em tempo real
- Alterar status do pedido
- Notificação sonora para novos pedidos
- Excluir pedidos

# ⚙️ Como rodar o projeto

Clone o projeto:

git clone https://github.com/Lpzin05/restaurant-order-system

Entre na pasta:

cd restaurant-order-system

Instale as dependências:

npm install

Execute o servidor:

node src/server.js

---

# 🌐 Acesse o sistema

Cliente:

http://localhost:3000/cliente.html

Painel:

http://localhost:3000/painel.html


⚠️ O servidor precisa estar rodando no terminal para que o sistema funcione.
